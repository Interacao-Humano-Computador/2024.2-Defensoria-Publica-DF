## Sobre o Projeto

Este repositório é dedicado à documentação e análise de Interação Humano-Computador (IHC) do site [Open Library](https://openlibrary.org/), realizada pelo Grupo 02 da disciplina de IHC na Faculdade do Gama da Universidade de Brasília (FGA-UnB), durante o 2º semestre de 2024.
